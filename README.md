# NvShaderCleaner

Программа для корректной полной очистки кэша шейдеров NVIDIA на Windows 10 / 11.

Делает то, что обычно приходится делать вручную — в правильной последовательности и с двумя нужными перезагрузками, между которыми сама себя продолжает.

---

## Что делает программа

1. Через **NVIDIA Profile Inspector** ставит «Размер кэша шейдеров» (Shader Cache Size) в **Disabled** в глобальном профиле NVIDIA.
2. Перезагружает компьютер (автоматически, с возможностью отмены в течение 5 секунд).
3. После входа в систему автоматически продолжает работу (через одноразовую задачу в Task Scheduler):
   - Спрашивает разрешение и закрывает фоновые процессы NVIDIA (включая NVIDIA Control Panel, GeForce Experience, ShadowPlay, nvcontainer и т. д.).
   - Удаляет папки `%LOCALAPPDATA%\NVIDIA\DXCache` и `%LOCALAPPDATA%\NVIDIA\GLCache`.
   - Находит установку Steam (включая дополнительные библиотеки на других дисках) и удаляет содержимое `…\steamapps\shadercache\730`. Если папки нет — уведомляет об этом.
4. Возвращает «Размер кэша шейдеров» в значение **Unlimited** (Без ограничений).
5. Перезагружает компьютер ещё раз.
6. После второй перезагрузки показывает красивое окно «Готово — кэш шейдеров успешно очищен».

Если на каком-то шаге что-то идёт не так — программа сообщает об этом отдельным окном с указанием этапа и пишет подробности в лог.

## Как собрать `.exe`

> Сборка выполняется на **Windows**. Нужен PowerShell 5.1+ (входит в Windows 10/11) и доступ в интернет (один раз — для установки `ps2exe`).

1. Скачайте/склонируйте этот репозиторий.
2. (Опционально) Положите рядом `icon.ico` — она будет использована как иконка `.exe`.
3. Запустите сборку **одним из двух способов**:

   ### Способ A — двойной клик (самый простой)

   Просто **двойным кликом** запустите файл `Build.cmd`. Откроется окно, в котором пройдёт сборка, в конце нужно нажать любую клавишу.

   ### Способ B — вручную через PowerShell

   Откройте PowerShell в папке репозитория. Введите **сначала одну команду, нажмите Enter, потом вторую**:

   Команда 1 (разрешает выполнение скриптов в текущем сеансе):

   ```powershell
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   ```

   Команда 2 (собственно сборка):

   ```powershell
   .\Build-Exe.ps1
   ```

   > ⚠️ Это **две отдельные команды**. Не копируйте их в одну строку — PowerShell склеит `Bypass` с `.\Build-Exe.ps1` и выдаст ошибку. Либо нажимайте Enter между ними, либо ставьте между ними точку с запятой: `... Bypass; .\Build-Exe.ps1`.
   >
   > Команда 1 вообще не обязательна — `Build-Exe.ps1` сам выставляет нужный режим для текущего процесса. Она нужна только если у вас совсем закручено `AllSigned`/`Restricted` и сам `.ps1` не запустится.

4. На выходе получите `NvShaderCleaner.exe` рядом со скриптом.

Готовый `.exe` уже содержит флаг `requireAdministrator`, поэтому при двойном клике сразу появится окно UAC.

## Как пользоваться

1. Поместите `NvShaderCleaner.exe` в **постоянную** папку (не во временный каталог и не на флешку, которую вы извлечёте — программа будет возвращаться к этому файлу после перезагрузок).
2. Запустите `NvShaderCleaner.exe` двойным кликом — Windows запросит права администратора.
3. При первом запуске программа скачает **NVIDIA Profile Inspector** (стабильная версия `2.4.0.4` с официального GitHub: `Orbmu2k/nvidiaProfileInspector`) в подпапку `tools\` рядом с собой.
4. Дальше просто следуйте инструкциям на экране. ПК автоматически перезагрузится два раза, после чего вы увидите финальное окно с сообщением об успехе.

### Если нет интернета

Если в момент первого запуска нет соединения с GitHub, программа подскажет, что нужно скачать `nvidiaProfileInspector.zip` (любую версию) вручную с
<https://github.com/Orbmu2k/nvidiaProfileInspector/releases>
и распаковать `nvidiaProfileInspector.exe` в папку `tools\` рядом с `NvShaderCleaner.exe`.

## Где хранится состояние и логи

- `%LOCALAPPDATA%\NvShaderCleaner\state.json` — текущий этап (после удачного завершения удаляется автоматически).
- `%LOCALAPPDATA%\NvShaderCleaner\log.txt` — подробный лог всех действий и ошибок.
- Запланированная задача `NvShaderCleaner_Resume` (Task Scheduler) — одноразовая, автоматически удаляется самой программой после завершения цикла.

Если вы хотите вручную «сбросить» состояние и начать заново — просто удалите `state.json` (и, на всякий случай, задачу `NvShaderCleaner_Resume` через `taskschd.msc`).

## Технические детали

- Изменение параметра `Shader Cache Size` выполняется командой:
  ```
  nvidiaProfileInspector.exe -setProfileSetting "Base Profile" 0x00E1C92E <value>
  ```
  где `value` = `0x00000000` (Disabled) или `0xFFFFFFFF` (Unlimited).
- Steam ищется через реестр (`HKCU\Software\Valve\Steam\SteamPath` и `HKLM\…\Valve\Steam\InstallPath`), а дополнительные библиотеки — из `libraryfolders.vdf`.
- Возобновление после перезагрузки реализовано через Task Scheduler (`AtLogon` + `Delay PT1M`, `RunLevel Highest`). Задача удаляется самим скриптом после выполнения.
- Авто-перезагрузка: `shutdown /r /t 5` с предварительным окном обратного отсчёта (5 секунд, кнопка «Отмена»).
- UI — WPF на встроенном XAML, тема в стиле NVIDIA (зелёный `#76B900` на тёмном фоне).

## Что делать, если что-то пошло не так

- Если очистка прервалась — просто запустите `NvShaderCleaner.exe` снова. Программа продолжит с того этапа, на котором остановилась (благодаря `state.json`).
- Если хочется начать «с чистого листа»: удалите `%LOCALAPPDATA%\NvShaderCleaner\state.json` и (если есть) запланированную задачу `NvShaderCleaner_Resume`.
- Подробности любых ошибок — в `%LOCALAPPDATA%\NvShaderCleaner\log.txt`.
